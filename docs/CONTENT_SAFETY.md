# CONTENT_SAFETY

## 자동 검증

운세/사주/저장/공유 관련 사용자 노출 문구는 `npm run check:content-safety`로 검증합니다.

검증 대상:

- 공통 안전 문구
- 공유 텍스트 생성 문구
- 사주 흐름 상세 화면 문구
- 오늘운세 상세 화면 문구
- 저장한 풀이 화면 문구
- 홈 화면 루틴/사주 흐름 안내 문구

검증 목적:

- 단정적 예언 표현 방지
- 건강/투자/법률/관계 결과를 확정하는 표현 방지
- 광고나 결제를 불안감으로 유도하는 표현 방지

주의:

- 문서 안의 금지 표현 예시는 검사 대상에서 제외합니다.
- 광고 로딩 실패 같은 시스템 오류 문구는 검사 대상에서 제외합니다.
- 실제 사용자에게 운세 해석으로 노출되는 문구를 우선 검사합니다.

## 목적

하루풀이의 운세, 사주 흐름, 저장한 풀이 콘텐츠가 참고용/엔터테인먼트 해석임을 사용자에게 일관되게 안내하기 위한 문서입니다.

## 공통 안전 문구 원칙

- 운세와 사주 해석은 참고용 흐름으로 표현합니다.
- 건강, 투자, 법률, 중대한 결정은 전문가 상담이나 현재 상황을 함께 고려하도록 안내합니다.
- 사용자의 불안을 자극하거나 행동을 강하게 몰아가는 문구를 사용하지 않습니다.
- 광고 해금 콘텐츠도 불안, 공포, 손실 회피를 이용해 열람을 유도하지 않습니다.

## 사용 위치

- HomePage: 홈 하단에 전체 서비스 공통 참고 안내를 표시합니다.
- FortuneDetailPage: 저장/공유 버튼 아래, 광고 해금 안내 근처에 오늘운세 참고 안내를 표시합니다.
- SajuInsightPage: 사주 흐름 해석 하단에 사주 해석 참고 안내를 표시합니다.
- SavedReadingsPage: 저장한 풀이 목록 위에 저장 시점의 참고용 해석임을 표시합니다.

## 금지 표현 예시

- 반드시 일어납니다.
- 무조건 실패합니다.
- 위험합니다.
- 이별합니다.
- 손해를 봅니다.
- 질병이 생깁니다.
- 지금 보지 않으면 불리합니다.
- 결제하지 않으면 운이 나빠집니다.

## 권장 표현 예시

- 참고용 해석입니다.
- 오늘의 흐름을 가볍게 살펴보세요.
- 현재 상황과 함께 고려해 주세요.
- 도움이 될 수 있습니다.
- 한 번 더 점검해보면 좋습니다.
- 속도를 조절해보세요.

## 전문 영역 안내

건강, 투자, 법률, 심리 위기, 중대한 계약이나 결정은 운세 결과만으로 판단하지 않도록 안내합니다. 필요한 경우 의료인, 법률 전문가, 재무 전문가 등 적절한 전문가 상담을 권장하는 톤을 유지합니다.

## 향후 작업

- 위험 표현 검증 스크립트를 추가해 운세 템플릿과 공유 문구를 점검합니다.
- 광고 해금 문구가 불안이나 과장에 기대지 않는지 주기적으로 확인합니다.
- 개인정보 처리방침과 광고 동의 UX가 준비되면 안전 문구 정책과 함께 검토합니다.
