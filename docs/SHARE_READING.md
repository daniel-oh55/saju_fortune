# SHARE_READING

## 목적

사용자가 오늘운세, 사주 흐름, 저장한 풀이를 외부 메신저나 메모 앱에 붙여넣기 쉽게 공유용 텍스트로 복사할 수 있게 한다.
현재는 서버 없이 브라우저 clipboard 기능만 사용한다.

## 공유 가능한 항목

- 오늘운세 요약
- 사주 흐름 무료 요약
- 저장한 풀이

## 공유하지 않는 항목

- 생년월일
- 출생시간
- 성별
- 음력/양력 원본 설정
- 23시 이후 자시 정책
- profile 원본 정보

## locked 상태의 상세 본문 공유 제한

- 오늘운세 상세가 광고 해금 전이면 상세 본문 전문을 공유하지 않는다.
- 공개된 요약, 행운 색상, 행운 아이템, 키워드 중심으로 공유한다.
- 광고 해금 심화 해석 전문 공유는 이번 범위에서 제외한다.

## 복사 방식

- 우선 `navigator.clipboard.writeText`를 사용한다.
- 브라우저가 clipboard API를 제공하지 않으면 `document.execCommand('copy')` fallback을 시도한다.
- 복사 실패 시 alert를 띄우지 않고 화면 안에 짧은 안내 문구를 표시한다.

## 문구 원칙

- 앱 이름 `하루풀리`를 포함한다.
- 공유 텍스트에는 참고용 해석이라는 문구를 포함한다.
- 단정적 예언처럼 보이는 표현을 피한다.
- 너무 길지 않게 정리하며 최대 길이는 약 1200자로 제한한다.

## 향후 확장

- 공유 이미지 생성 기능을 별도 PR에서 검토할 수 있다.
- 앱 설치 이후에는 OS 네이티브 공유 API 연동을 검토할 수 있다.
- 공유 텍스트 템플릿은 브랜드/콘텐츠 가이드에 맞춰 계속 보완한다.
