하루풀이 개인정보 처리방침

시행일: 2026년 7월 12일

제공자/운영자: 하루풀이 운영자

1. 개인정보 처리방침 목적

이 개인정보 처리방침은 하루풀이 앱의 개인정보 처리 기준을 설명합니다. 하루풀이는 운세/사주 콘텐츠 제공을 위해 사용자가 입력하는 정보를 활용하며, 해당 정보는 기기 내 저장 구조를 중심으로 관리됩니다.

2. 수집 또는 입력될 수 있는 정보

하루풀이 이용 과정에서 아래 정보가 입력되거나 저장될 수 있습니다.

  • 생년월일
  • 출생시간
  • 출생지역
  • 성별
  • 사용자가 입력한 프로필 정보
  • 저장한 운세/풀이 기록
  • 앱 사용 중 사용자가 직접 저장한 데이터

3. 개인정보 저장 방식

  • 현재 하루풀이는 서버 DB를 사용하지 않습니다.
  • 현재 하루풀이는 로그인 기능을 사용하지 않습니다.
  • 현재 하루풀이는 사용자가 입력한 정보와 저장한 풀이를 기기 내 저장소 중심으로 보관합니다.
  • 웹앱 환경에서는 브라우저 localStorage 등 기기 내 저장소를 사용할 수 있습니다.
  • Android 앱 환경에서도 앱 내부 저장 영역 또는 WebView 저장소에 데이터가 남을 수 있습니다.

4. 개인정보 이용 목적

  • 오늘의 운세 제공
  • 나의 사주 흐름 제공
  • 2026 운세 제공
  • 띠별운세 제공
  • 저장한 풀이 다시 보기
  • 사용자가 입력한 정보를 바탕으로 한 운세 콘텐츠 표시

5. 제3자 제공

현재 하루풀이는 사용자가 입력한 개인정보를 별도 서버 DB로 저장하거나 제3자에게 판매하지 않습니다.

현재 하루풀이는 실제 광고 SDK, 결제 SDK, 외부 분석 SDK를 사용하지 않습니다.

향후 광고, 결제, 분석 기능이 추가되는 경우 개인정보 처리방침을 업데이트합니다.

6. 개인정보 보관 및 삭제

  • 사용자가 앱 내 저장된 풀이를 삭제할 수 있는 경우 해당 기능을 통해 삭제할 수 있습니다.
  • 브라우저 또는 앱 저장 데이터 삭제를 통해 기기 내 저장 데이터를 삭제할 수 있습니다.
  • 앱 삭제 시 기기 내 앱 데이터가 함께 삭제될 수 있습니다.
  • 단, 운영체제/브라우저 정책에 따라 삭제 방식이 다를 수 있습니다.

7. 아동의 개인정보

  • 하루풀이는 별도의 회원가입 없이 운세 콘텐츠를 제공하는 앱입니다.
  • 법정대리인 동의가 필요한 연령의 사용자는 보호자와 함께 이용하는 것을 권장합니다.
  • 민감한 개인정보를 입력하지 않도록 안내합니다.

8. 이용자 권리

  • 사용자는 본인이 입력한 정보를 수정하거나 삭제할 수 있습니다.
  • 앱 저장 데이터 삭제 또는 앱 삭제를 통해 기기 내 데이터를 삭제할 수 있습니다.
  • 문의사항은 support.hym@gmail.com으로 연락할 수 있습니다.
