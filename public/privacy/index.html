하루풀리 개인정보 처리방침

최종 수정일: 2026-06-16

이 문서는 하루풀리의 Google Play 제출을 위한 공개 개인정보 처리방침 후보 페이지입니다. 현재 MVP 기준으로 작성되었으며, 실제 광고 SDK, 결제 SDK, 로그인, 서버 DB가 추가되면 갱신해야 합니다.

현재 이 페이지는 Google Play 제출 전 확인용 scaffold입니다. 실제 최종 URL은 Vercel 배포 후 확인합니다.

1. 서비스명

서비스명은 하루풀리입니다.

2. 수집 또는 저장하는 정보

하루풀리는 현재 MVP에서 사용자의 기기 안에 다음 정보를 저장할 수 있습니다.

  • 사용자가 입력한 프로필 정보
  • 오늘의 운세 cache
  • 사주 분석 cache
  • 보상 unlock 상태
  • 동의 설정
  • 저장한 운세
  • 방문 streak

3. 저장 위치

현재 MVP는 서버 데이터베이스 없이 사용자의 브라우저 또는 Android WebView localStorage를 중심으로 정보를 저장합니다. 현재 기준으로 위 정보는 서버로 전송하지 않습니다.

4. 이용 목적

  • 오늘의 운세 및 사주 흐름 계산
  • 동일 날짜 결과 유지
  • 저장한 운세 다시 보기
  • 방문 streak 표시
  • 동의 설정 관리
  • 보상형 상세 콘텐츠 unlock 상태 관리

5. 외부 제공 및 제3자 공유

현재 MVP에서는 제3자에게 데이터를 공유하지 않습니다.

현재 실제 광고 SDK를 사용하지 않습니다.

현재 실제 결제 SDK를 사용하지 않습니다.

현재 실제 외부 분석 SDK는 연결되어 있지 않습니다.

향후 SDK가 추가되면 개인정보 처리방침과 Google Play 데이터 보안 양식을 함께 갱신해야 합니다.

6. 광고 및 분석

현재 실제 광고 SDK는 연결되어 있지 않습니다.

현재 실제 외부 분석 SDK는 연결되어 있지 않습니다.

앱에는 향후 보상형 광고 또는 광고 표시를 위한 UI와 구조가 준비될 수 있습니다. 실제 광고 SDK를 연결하는 경우 이 문서를 갱신합니다.

7. 결제

현재 실제 결제 SDK는 연결되어 있지 않습니다.

유료 기능이 추가되면 결제 관련 개인정보 처리 항목을 갱신해야 합니다.

8. 사용자 데이터 삭제

앱 설정 또는 브라우저/앱 데이터 삭제를 통해 localStorage 데이터를 초기화할 수 있습니다. Android 앱에서는 앱 정보 > 저장공간 > 데이터 삭제 또는 adb shell pm clear com.harupuli.app 명령으로 초기화할 수 있습니다.

현재 서버 DB가 없으므로 서버 데이터 삭제 요청 대상은 없습니다.

9. 참고용 콘텐츠 고지

하루풀리의 운세와 사주 결과는 오락 및 참고용 콘텐츠입니다. 의학, 법률, 투자, 진로, 심리 상담 등 전문적인 자문을 대체하지 않습니다.

10. 문의처

문의처는 Google Play 제출 전 확정 필요 상태입니다.
